The color section is the most powerful layoutÂ element in your arsenal to build unique websites. it allows you to display fullwidth section that contains various elements, and also lets you set the styling of this section by changing background images and color, height, top and bottom border, etc

A color section is a fullwidth element that can not co-exist beside a sidebar. You can however display a sidebar below the last fullwidth element if you want to.
